LabelMaker
by Jeff Conner

Instructions:

1.  Populate a file called input.txt  with "x y z <label_name>" values and place it in data\
2.  run the code
3.  The code places a box around the perimeter of the values in the input file.
4.  Press "delete" to remove the box and "insert" to put it back in
5.  Press "enter" to export the scene to an osg file called "output.osg"
